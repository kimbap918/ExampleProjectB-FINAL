# 🛒 쇼핑몰 프로젝트 SME



## 📄 개발 배경

쇼핑몰에서 주문 접수와 판매를 진행하다 보면, 자료의 취합이 어려워 재고관리를 포기하는 경우가 많다.

SME 쇼핑몰 프로젝트에서는 재고관리와 ERP를 연동한 웹 쇼핑몰을 개발해 쇼핑몰에 산재한 재고관리 데이터를 통합하여 확인하고 관리할 수 있게 했다.
